# 🌱 HabitFlow – Smart Habit Tracker

A modern, responsive habit tracking web app built with **React** and **Recharts**.  
Track your daily habits, visualize progress, and build consistency with ease.

---

## ✨ Features

- ✅ Add, Edit, Delete habits
- 📅 Monthly habit tracking grid
- 📊 Interactive charts (Daily + Weekly)
- 🔥 Streak tracking system
- 🏆 Top habits ranking
- 📈 Progress visualization with circular indicators
- 🌙 Dark / Light mode toggle
- 💾 LocalStorage support (data persists)
- 🎯 Clean & responsive UI
